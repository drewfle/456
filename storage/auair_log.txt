Turbo boost doesn't exceed base frequency 1.7 GHz
1. [useful dir contains each core's config: /sys/devices/system/cpu/cpu0/cpufreq]
2. [try edit grub at /etc/default/grub, bacicall it won't work in ubuntu 14.01.1]
2.1 GRUB_CMDLINE_LINUX_DEFAULT="processor.ignore_ppc=1"
2.2 # update-grub
3. [Solution 2: include the following script in rc.local]
3.1 [create a file to store the script]
3.1.1 # vim /etc/init.d/99increfreq
3.1.2 [paste the script, by setting maximum frequency at 230100, the cpu will automatically go beyond 2.3 GHz to its designed turbo booost frequency at 2.9/3.3]
#
echo 1 | sudo tee /sys/module/processor/parameters/ignore_ppc
for x in /sys/devices/system/cpu/cpu[0-3]/cpufreq/ ; do
        echo 2301000 > $x/scaling_max_freq;
done

#
3.1.4 # chmod +x vim /etc/init.d/99increfreq
